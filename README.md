# 👋

I'm Aodhan. Embedded systems engineer building scientific instruments with FPGAs, custom hardware, and NixOS.

## 🔧 What I do

- **FPGA** — custom Verilog/SystemVerilog for signal processing and precision timing.
- **Embedded hardware & firmware** — PCB design, bring-up, bare-metal and RTOS.
- **Scientific instrumentation** — building custom spectroscopy and measurement systems at [@solarpi-org](https://github.com/solarpi-org).
- **Data & monitoring** — Python, data pipelines, and dashboards for solar energy systems.
- **Nix** — declarative infrastructure, dev environments, and deployment from cloud to embedded.
- **Test & measurement** — oscilloscopes, logic analyzers, ADCs, instrument automation.
